# 关灯游戏

这是一个简单的关灯游戏，使用Python开发。

## 游戏介绍

在本游戏中，玩家需要通过点击屏幕上的灯泡来改变其状态（亮/灭），目标是通过最少的点击次数将所有灯泡熄灭。

## 游戏规则

- 点击一个灯泡会改变它周围（上、下、左、右）的灯泡状态。
- 游戏开始时所有灯泡都是亮的。
- 玩家需要通过点击灯泡，最终使所有灯泡熄灭。

- **编程语言**：Python
- **框架/库** ：Pygame等
- **开发环境**：PyCharm
